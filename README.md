### What I'm building

- **[nfc-pos-system](https://github.com/DilanProjects1001/nfc-pos-system)** — End-to-end NFC payment + virtual-currency POS for a real club. A Proxmark3 daemon streams card UIDs over WebSocket; backend enforces anti-double-charge locks and `balance >= 0` integrity.
